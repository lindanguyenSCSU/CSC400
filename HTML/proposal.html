Hotel Reservation - Vtrip

Motivation.

People all over the world are fond of travel and tourism. People often find it difficult to search for the best places. To address the issue , we adopt the travelling website which will offer the best places among others. Hotels are much larger now with most of the people going online, searching deep for information about where they want to stay, facility and pricing all together will create a business opportunity for owners to attract more visitors. Especially, during pandemic time, most people would choose the option of booking and checking online which will save a lot of time and it is a good idea for safety issues too, so online would be the best solution for this situation. This project is based on how an effective travel website will make us feel. It is based on a travelling website with fully featured functions that will activate the travelling bug with vibrant imagery. This website contains highlights of some important places along with high quality photography and allows people to book their dream destination within their budgets.

Project Objection

This application will provide service to on-line customers looking for a place. Online customers can make searches, reservations and cancel an existing one. The project will keep track of Guest details, Room Reservation, Room Booking.

Features

  • Users can search for a hotel in a given city or state
  • System will display all available hotel, motels
  • Users can make reservations, book and cancellations via online
  • Each time a customer makes a reservation, room status is updated and the customer details are added to the hotel database.
  • Administrator can add/update the hotel

Technical requirement

  • Operating System (Window, MAC, Linux, ..)
  • GitHub as the repository for storage of relevant data
  • Coding with the use of PHP, MYSQL, JAVASCRIP, HTML, CSS
